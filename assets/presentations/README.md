# Presentation Files

AI 프로젝트 발표 자료를 이 폴더에 넣고 `manifest.json`에 등록하면 포트폴리오 안에서 바로 열립니다.

PDF는 PDF Viewer로 열리고, HTML 발표 자료는 Browser.app 안에서 열립니다.

현재 등록된 기본 파일명:

- `emotion-style-transfer.pdf`
- `clip-blip2-retrieval.html`
- `dueling-ddqn-yakemon-agent.pdf`
- `qcnn.pdf`

파일명이 다르면 `manifest.json`의 `file` 값을 실제 파일명으로 바꾸면 됩니다.
