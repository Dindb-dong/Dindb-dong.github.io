Project Logic

Core Workflow

1
Query Analysis — Gemini
한국어 쿼리 → 최대 2개 영어 동작 문장 분할 / API 실패 시 rule-based fallback
2
Window Search — CLIP
Sliding Window p_sec / step_sec → Quick sampling q/3 → Top-K 필터 미달 window 스킵
3
Sequence Localization — Top-K anchor
상위 K window를 anchor로 선택 → 좌/우 경계 탐색 (Bidirectional) → 다중 하락 후보 비교
4
EMA Refinement — 경계 정밀화
전역 유사도 타임라인 → EMA 기반 구간 수렴 → 2쿼리 분할 시 합집합으로 최종 구간 결정
5
Output
JSON 결과 + Window/Frame 점수 + 실시간 시각화 + EMA 분석 리포트
