# 썸네일 이미지 폴더

이 폴더에는 프로젝트 썸네일 이미지를 저장합니다.

## 파일 명명 규칙

- `proj_{프로젝트ID}_{이미지번호}.png` 또는 `.jpg`
- 예시:
  - `proj_1_1.png` - 첫 번째 프로젝트의 첫 번째 이미지
  - `proj_1_2.jpg` - 첫 번째 프로젝트의 두 번째 이미지
  - `proj_2_1.png` - 두 번째 프로젝트의 첫 번째 이미지

## 사용 방법

1. 프로젝트 썸네일 이미지를 이 폴더에 저장하세요.
2. 파일명은 위의 규칙에 따라 명명하세요.
3. PNG 또는 JPG 형식을 사용할 수 있습니다.
4. 갤러리에서는 각 프로젝트의 첫 번째 이미지(`proj_{id}_1`)가 표시됩니다.
5. 상세 페이지에서는 해당 프로젝트의 모든 이미지가 표시됩니다.
